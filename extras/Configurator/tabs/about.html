Phoenix Configurator was designed to improve user experience by not requiring extensive programming knowledge
needed to do proper sensor calibration and debugging of various flight control elements (kinematics, motor outputs, etc).

This also allows users to do PID tuning right from the configurator (without the need to reflash the flight controller afterwards).
Because all of the configuration changes are executed on the flight controllers side in real time.
